# Screenshots para PWA

Esta carpeta debe contener screenshots de la aplicación para PWABuilder:

## Archivos necesarios:

1. **screenshot-wide.png** (1280x720px)
   - Captura de pantalla del dashboard en escritorio
   - Formato: PNG

2. **screenshot-mobile.png** (390x844px)  
   - Captura de pantalla del dashboard en móvil
   - Formato: PNG

## Cómo crear los screenshots:

### Opción 1: Manualmente
1. Abre la app en el navegador
2. Usa las DevTools (F12) para simular móvil (iPhone 14 Pro)
3. Toma una captura con la herramienta de recorte
4. Ajusta al tamaño requerido

### Opción 2: Con Chrome DevTools
1. F12 → Device Toolbar (Ctrl+Shift+M)
2. Selecciona resolución personalizada
3. Menú ⋮ → Capture screenshot

### Nota:
Los screenshots son opcionales para generar el APK, pero recomendados para mejor puntuación en PWABuilder.

# Descargas - Mi Horario FPUNA

## APK para Android

El archivo `mi-horario-fpuna.apk` se colocará aquí una vez generado.

### Cómo generar el APK

1. Ve a [PWABuilder](https://www.pwabuilder.com/)
2. Ingresa la URL: `https://alfarot.github.io/Mi-Horario-FPUNA/`
3. Selecciona "Android" y descarga el paquete
4. Renombra el APK a `mi-horario-fpuna.apk`
5. Colócalo en esta carpeta

### Notas

- El APK es un contenedor TWA (Trusted Web Activity)
- Se actualiza automáticamente cuando actualices la web
- Tamaño aproximado: 2-3 MB
